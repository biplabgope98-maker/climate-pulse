# 🌦 Climate Pulse

A premium weather dashboard built using **HTML, CSS, JavaScript, jQuery, and OpenWeather API**.

## 🚀 Live Demo

https://biplabgope98-maker.github.io/climate-pulse/

## ✨ Features

- 🌙 Dark / ☀️ Light Theme Toggle
- 🌍 Real-Time Weather Updates
- 🔍 Search Weather by City
- 💨 Wind Speed Information
- 💧 Humidity Details
- 🌡 Feels Like Temperature
- 🕒 Recent Search History using LocalStorage
- 📱 Responsive Dashboard UI
- 🎨 Premium Glassmorphism Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- LocalStorage
